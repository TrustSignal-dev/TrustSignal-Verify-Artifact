# Repository Guidelines

## Purpose

This repository is deprecated and retained for history only. The canonical GitHub Action now lives in `TrustSignal/github-actions/trustsignal-verify-artifact`.

## Guardrails

- Do not ship new releases from this repository.
- Do not add new product behavior here.
- Prefer documentation changes that point users to the canonical action.
- Do not let this repo drift back into being treated as the source of truth.

## Allowed work

- Clarify deprecation status.
- Improve migration guidance.
- Preserve historical context where useful.

## Not allowed

- New runtime features
- New integrations
- New release tags or packaging intended for production use
